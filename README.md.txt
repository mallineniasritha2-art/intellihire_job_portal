# IntelliHire – Online Job Recruitment System

IntelliHire is a web-based job recruitment platform developed using Django. 
The system allows candidates to apply for jobs and interviewers to manage 
applications and schedule interviews.

## Features
- User authentication (Signup/Login)
- Role-based dashboards (Candidate & Interviewer)
- Job posting by interviewer
- Job application system for candidates
- Accept/Reject application management
- Interview scheduling with date and time
- Admin panel for system management

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run the Project

1. Clone the repository
2. Create virtual environment
3. Install dependencies
